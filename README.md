本项目由纯java代码开发，核心代码为pcqq协议，翻译自网络上其他开源的pcqq协议代码，使用本项目代码产生的任何后果与本人无关。
### 功能:
  静态代码类:
    登录qq，获取群列表/好友列表，发送接受群/好友消息，已插件化，可用java/lua插件
  动态代码类:
    可用tui实现简单聊天
